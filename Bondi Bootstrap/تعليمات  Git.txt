 _______New Lession 8________
 تخزين ملفات في خزنة خاص بالغيت وعدم اظهارها في موقع او ستاتيوس
1-$ git stash 
تخزين ملفات في ستاش بعد عملية ستاتيوس
2-$ git stash list
اظهار ملفات مخفية عندك في ستاش 
3-$ git stash pop
جلب بيانات من ستاش واظهارهم بين ملفات ستاتيوس .وحذف ملف من ستاش ثم بعد ذلك نقوم بالعملية بوش 
$ git stash save "ppo"
حفظ ملف باسم معين في ستاش 
$ git stash apply
جلب بيانات من ملف ستاش وعدم حذف ملف من ستاش وبعجها نقوم بالعملية بوش
$ git stash pop stah@{0}
جلب بيانات ثم حذف ملف من ستاش ذو معرف 0 
$ git stash drop
حذف ملف من ستاش بدون جلب بيانات 
$ git stash clear
حذف جميع ملفات من ستاش بدون جلب بيانات 
$ git stash show
عرض محتويات مجلد ما في ستاش 
ملاحظة :  عند عملية حذف وعرض وعدم تعريف نةع معرف  يقوم بجلب اخر ملف اتم اضافته الى ستاش
